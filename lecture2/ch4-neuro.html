第4章：神経科学との接続 ― なぜ潮吹きは起こるのか

REVIEW

4つの神経経路 ― 講義1の復習

講義1で学んだ4つの神経経路。これが潮吹きのメカニズムにも直結します。

女性の性器→脳 4つの神経経路と潮吹きの関係

MECHANISM

潮吹きの反射メカニズム

潮吹きがオーガズムと独立して起こりうる理由は、反射経路の違いにあります。

オーガズムの経路

複数の神経 → 脳の報酬系 → 意識的な快感

CUV複合体の刺激 → 骨盤・陰部・迷走神経 → NTS孤束核。脳幹にある中継基地。迷走神経からの信号を受け取り、視床下部に転送。オキシトシンやドーパミンの放出を促す。/視床下部 → ドーパミン・オキシトシン大量放出 → 意識的な快感体験

潮吹きの経路

膀胱周辺の刺激 → 脊髄反射 → 不随意的な排出

CUV複合体の刺激 → 膀胱頸部・尿道括約筋の弛緩 → 骨盤底筋の収縮 → 膀胱内容物の不随意的排出

→ 脳の快感中枢を経由しない場合がある

だから「快感なし」の潮吹きがある

オーガズムは脳の報酬系を経由する意識的体験。潮吹きは脊髄反射レベルで起こりうる不随意反応。
二つの回路が同時に発火すれば「快感を伴う潮吹き」になり、
潮吹き回路だけが発火すれば「快感のない潮吹き＝構造上出てしまうだけ」になる。

これが「快感の人もいれば不快な人もいる」の神経科学的な説明です。

CUV複合体の刺激 ＝ 複数の神経が同時発火

膣前壁エリア（旧Gスポット領域）を刺激すると、CUV複合体を構成する複数の構造が同時に刺激されます。

クリトリス内部（脚部）の刺激

クリトリスの大部分は体内にある。膣前壁を通じて内部のクリトリス脚部が圧迫される → 陰部神経が発火

尿道・スキーン腺の刺激

膣前壁の直上に尿道とスキーン腺がある。圧迫により充血・分泌が促進 → 骨盤神経が発火＋Female Ejaculation

膀胱頸部への波及

強い刺激が膀胱頸部に波及 → 尿道括約筋の弛緩 → 下腹神経の反射＋Squirting

子宮頸部（ポルチオ）への伝達

深い刺激が子宮頸部に到達 → 迷走神経が発火 → NTSを介して脳に直接

講義1との接続

講義1で学んだ迷走神経のバイパス経路は、潮吹きよりもむしろオーガズムの質に関与。
潮吹き自体は主に脊髄反射（骨盤・下腹神経）で起こるが、
迷走神経が同時に発火すると快感を伴った深い体験になる。

つまり、ゴリ式の体外ポルチオアプローチで迷走神経を先に活性化しておくことで、
潮吹きが起きた場合もそれが快感を伴う体験になりやすい。

未解明の謎：膀胱の急速充填

性的興奮中に、排尿直後の空の膀胱が急速に再充填される現象のメカニズムは2024年時点でもまだ完全には解明されていません。

仮説1：腎臓の血流増加

性的興奮時の血圧上昇 → 腎血流増加 → 尿生成の加速。ただし通常このスピードでは尿は作れない。

仮説2：血漿の浸出

骨盤内の血管充血 → 血漿成分が膀胱壁を通じて浸出。通常の尿より薄い理由を説明できる可能性。

仮説3：自律神経の関与

迷走神経・交感神経の複雑な相互作用が膀胱の充填を促進している可能性。研究が進行中。

科学のフロンティア

「なぜ起こるのか」の最終的な答えはまだ出ていません。しかし「何が起こっているか」は超音波と色素実験で明らかになっています。科学は完璧な答えを持っていないが、確実に前進している。この誠実さこそが、受講生の説得力になります。
